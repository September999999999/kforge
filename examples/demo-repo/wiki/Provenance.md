# Provenance

Provenance is the link between a synthesized wiki statement and the evidence
that supports it.

In a kforge repo, local source references make the knowledge base auditable by
deterministic tools. `kforge doctor` can check whether wiki pages, claims, and
reviews point to source files that actually exist.
